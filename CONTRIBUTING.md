# Contributing to Composer Galaxy

Thank you for helping improve Composer Galaxy.

Composer Galaxy is an exploratory, source-linked public beta that maps teacher–student relationships between composers. The goal is to make the map more accurate, useful, inclusive, and transparent over time.

You do **not** need to know how to code to contribute.

## Ways to contribute

You can help by suggesting:

- missing composers
- incorrect teacher–student relationships
- missing teachers or students
- better source links or citations
- corrections to birth/death dates, names, or metadata
- improvements to the interface or visual design
- bugs or confusing behavior in the app

## Reporting an incorrect relationship

If you see a relationship that looks wrong, please include:

- the composer names involved
- what the app currently says
- what you think should be changed
- a source, if you have one

Example:

> The app lists Composer A as a student of Composer B, but I think this may be incorrect. This source says they studied at the same institution but does not describe a direct teacher–student relationship: [link]

## Suggesting a missing composer

Please include as much of the following as you can:

- composer name
- birth/death dates, if known
- Wikipedia page or other reliable source
- known teachers
- known students
- why you think they should be included

Composer Galaxy currently focuses mostly on written/notated music, broadly defined. The project has a particular interest in improving representation of women composers, composers of color, and historically underrepresented musical lineages.

## Suggesting better sources

Better sources are always welcome.

Useful sources include:

- composer biographies
- Grove/Oxford Music references
- institutional biographies
- program notes
- scholarly articles
- archival sources
- reliable encyclopedia entries
- composer or foundation websites

When possible, please include a direct link and a short note explaining what the source supports.

## Data reliability

Composer Galaxy uses an LLM-assisted extraction pipeline to identify possible teacher–student relationships from source text. Some relationships are confirmed by cross-reference or Wikidata; others are marked as less certain.

Please do not assume every edge is definitive. The source and confidence markers are part of the project.

## Code contributions

Code contributions are welcome too.

Helpful areas include:

- interface improvements
- mobile layout fixes
- accessibility improvements
- graph layout improvements
- better contribution/flagging workflows
- data validation tools
- documentation improvements

## How to submit feedback

The easiest way to contribute is to open a GitHub issue.

When in doubt, open an issue and describe what you noticed. Partial information is still useful.
